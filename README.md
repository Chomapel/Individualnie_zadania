# Задания по C++

Здесь собраны простые программы на C++:

1. Двумерный массив float со случайными числами
2. Двумерный массив int со случайными числами
3. Машинный ноль и машинный эпсилон
4. Длина вектора
5. Поиск простых чисел
6. Счастливые билеты
7. Числа-перевертыши
8. Максимум функции
9. Умножение матриц
10. Трёхзначные числа с заданной суммой цифр

Каждая программа компилируется и запускается отдельно.
